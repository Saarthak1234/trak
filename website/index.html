Why Musync?

🚀

Zero-Setup Playback

Paste any public Spotify URL to start playing instantly. No accounts, no API keys, no configuration.

🔐

Optional Login

Connect your Spotify account to seamlessly access and search through your private saved playlists.

⌨️

Command Mode

Full keyboard support. Hit / during playback to instantly jump to any track or search for a custom song.

📦

Auto Dependencies

The built-in wizard automatically installs ffmpeg and yt-dlp for you under the hood.

Installation

The easiest way to install Musync on Mac or Linux is using our one-line auto-installer. Open your terminal and run:

curl -sL https://raw.githubusercontent.com/Saarthak1234/musync/main/install.sh | bash

This will automatically download the correct binary for your system and make it globally available. If you are on Windows, download the executable directly from the GitHub Releases page.

Usage Manual

1. Play Public URLs (No Login Required)

Play any public playlist instantly:

musync play "https://open.spotify.com/playlist/..."

2. Search & Play Single Songs

Find a specific track instantly:

musync search "Rick Astley"

3. Authenticate with Spotify

If you want to play your private playlists by name, you can securely log in via OAuth:

musync auth
musync list
musync play "My Awesome Playlist"
